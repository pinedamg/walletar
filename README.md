# WalletAR

WalletAR es una aplicación móvil para la gestión de finanzas personales. Proporciona herramientas para registrar ingresos, gastos, transferencias y realizar reportes financieros, todo integrado con Firebase.

---

## Características Principales

- **Gestión de cuentas:**  
  Soporte para cuentas de diferentes tipos como efectivo, tarjetas de crédito y cuentas bancarias.
  
- **Gestión de transacciones:**  
  Registra ingresos, gastos y transferencias con soporte para monedas múltiples.
  
- **Autenticación segura:**  
  Inicia sesión con correo electrónico, contraseña y autenticación biométrica (huella digital).

- **Reportes financieros:**  
  Genera reportes detallados para analizar gastos e ingresos por categoría.

---

## Tecnologías Utilizadas

- **Flutter**: Framework para desarrollar aplicaciones móviles en Android e iOS.
- **Firebase**:
  - Authentication.
  - Cloud Firestore.
  - Firebase Cloud Functions.
